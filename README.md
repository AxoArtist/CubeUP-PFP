# CubeUP-PFP
Ever wanted to make a cubeUP pfp now you can!!!

also you can find youtube tut here: NOTHING HERE YET :(

WHAT YOU NEED:
-blender

steps:



1. just load up the blend file
2. click on player
3. click on all the cubes ALSO THE HANDS
4. go to material
5. add a material
6. choose your color
7. click on the cosmetics collection
8. click on hat / hands / face
9. click on the eyeball with the cosmetic you want
10. press numpad 0
11. open any screen picture program
12. take the picture
13. put it in photopea (make sure to select 512 / 512)
14. save the PNG
15. put it as your PFP
